# rig-metrics-aggregator

* How to run
```shell
export APP_INSIGHTS_INS_KEY=<instrumentationKey>
export RIG_0_NAME=<rig name>
export RIG_0_HOST=<rig ip or hostname>
export RIG_0_PORT=<xmrstak port>
node index.js
```
